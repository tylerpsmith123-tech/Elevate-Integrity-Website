# Elevate Integrity Services Website

This is a Vercel-ready React + Vite + Tailwind website.

## Deploy on Vercel

1. Download and unzip this folder.
2. Put the folder on your desktop.
3. Go to https://vercel.com/new.
4. Choose **Add New Project**.
5. If using GitHub, upload this folder to a new GitHub repository first, then import it.
6. Framework preset should auto-detect as **Vite**.
7. Build command: `npm run build`
8. Output directory: `dist`
9. Click **Deploy**.

## Edit Contact Info

Open `src/App.jsx` and replace:
- `(000) 000-0000`
- `info@elevateintegrity.com`

## Local Preview

Run:
```bash
npm install
npm run dev
```
